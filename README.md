# Portfolio 2026 - [Marjorie]

Ce portfolio a été conçu avec un objectif central : **l'efficacité** et **Rapidité**. J'ai privilégié une approche légère et performante
pour démontrer ma maîtrise des fondamentaux du Web.

## 🚀 Philosophie du projet
Pour ce projet, j'ai fait le choix de ne pas utiliser de frameworks complexes (comme React ou Next.js). 
* **Rapidité :** Chargement quasi-instantané grâce au HTML pur et Vanilla JavaScript.
* **Légèreté :** Zéro dépendance superflue en production.
* **Design :** Utilisation de **Tailwind CSS 4** pour une interface moderne, soignée et entièrement responsive.

## 🛠️ Stack Technique
* **Langages :** HTML5, JavaScript (ES6+)
* **Style :** Tailwind CSS 4 (Configuration personnalisée via `@theme`)
* **Outils :** Git, GitHub, GitHub Pages

## ♿ Roadmap & Accessibilité (V2)
Ce projet est en constante évolution. La prochaine étape (V2) sera dédiée à l'**inclusion et l'accessibilité** :
- [ ] Intégration d'un **Dark Mode** natif.
- [ ] Optimisation des contrastes et de la sémantique HTML.
- [ ] Ajout d'une typographie spécifique pour faciliter la lecture (ex: OpenDyslexic, Luciole ou renforcement du confort visuel).

## 📂 Structure du projet
- `/images` : Assets et logos.
- /CSS : Fichiers sources et configuration @theme de Tailwind. 
- /dist : CSS compilé et optimisé pour la production.
- `/js` : fonctions utilitaires.
- `/pages` : Structure HTML.
- `index.html` : Point d'entrée principal.
